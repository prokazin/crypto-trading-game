# Crypto Trading Game

Реалистичная трейдинг игра в стиле Bybit с интеграцией Telegram Web Apps.

## Особенности

- 📊 Реалистичный график цен в стиле Bybit
- 💰 Начальный баланс: $2,000
- 🪙 3 криптовалюты: BTC, SHIB, DOGE
- 📈 Кредитное плечо: 1x, 2x, 5x, 10x
- ⚡ Моментальное открытие/закрытие позиций
- 📱 Полная мобильная адаптация
- 📉 Автоматическая ликвидация при недостатке маржи
- 📊 Влияние торгов на рыночную цену
- 📈 Динамическое изменение спреда
- 📰 30 рыночных событий (15 положительных, 15 отрицательных)
- ⭐ Покупка Telegram Stars за реальные деньги
- 💰 Конвертация звезд в игровые доллары
- 👥 Рейтинг игроков
- 📱 Нативная интеграция с Telegram Web Apps
- 💾 Автосохранение в localStorage
- ☁️ Синхронизация через Telegram Cloud Storage
